## 🆕 v0.2.0 新功能一览

### 云端备份 API

```bash
# 上传备份
POST /backup/upload
{
  "deviceId": "phone1",
  "backup": {
    "tasks": [...],
    "exportedAt": "2026-03-13T10:00:00"
  }
}

# 列出备份
GET /backup/list?deviceId=phone1

# 下载备份
GET /backup/{id}/download

# 删除备份
DELETE /backup/{id}
```

### 核心特性
- ✅ 每个设备最多 5 个备份
- ✅ 超出自动删除最旧的
- ✅ 支持任意 JSON 备份数据
- ✅ 自动计算任务数和大小

## 🆘 常见问题快速排查

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| 编译超时 | 网络问题 | 脚本会自动重试 3 次 |
| Docker 构建失败 | 网络/磁盘 | 增加超时或清理磁盘 |
| 应用启动失败 | 数据库连接 | 检查 application.yaml |
| 无法访问 API | 防火墙 | 检查云服务商安全组 |
